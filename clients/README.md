# Clientes

Esta carpeta sirve para documentar cada cliente nuevo que se active en el sistema.

## Convención
Cada cliente puede tener un archivo propio, por ejemplo:

- ideas-graficas.md
- cliente-b.md
- cliente-c.md

## Contenido sugerido
- nombre del cliente
- proyecto Vercel
- proyecto Supabase
- URL
- admin inicial
- módulos habilitados
- fecha de alta
