# Alta de nuevo cliente

## Objetivo
Crear una nueva app y base de datos independiente para un cliente nuevo.

## Pasos
1. Crear proyecto Supabase del cliente
2. Aplicar migraciones
3. Ejecutar seeds base
4. Crear proyecto Vercel del cliente
5. Configurar variables de entorno
6. Crear usuario admin inicial
7. Probar login
8. Probar ventas
9. Probar inventario
10. Entregar URL al cliente
