# Arquitectura del sistema

## Producto base
Sistema web de punto de venta para negocio de comida.

## Stack
- Next.js
- Vercel
- Supabase
- Tailwind CSS
- TypeScript

## Módulos principales
- Login y roles
- Productos
- Materia prima
- Recetas
- Ventas / POS
- Cocina
- Compras a proveedor
- Inventario
- Caja y cortes
- Reportes

## Roles
- admin
- cajero
- cocina
- almacen

## Estrategia de clientes
Una app por cliente y una base de datos por cliente.

## Ambientes
- dev
- stg
