# Variables de entorno

## Variables mínimas
- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY

## Variables de configuración por cliente
- NEXT_PUBLIC_APP_NAME
- NEXT_PUBLIC_CLIENT_NAME
- NEXT_PUBLIC_SUPPORT_EMAIL
- NEXT_PUBLIC_CURRENCY
- NEXT_PUBLIC_TIMEZONE
- NEXT_PUBLIC_ENABLE_KITCHEN
- NEXT_PUBLIC_ENABLE_INVENTORY

## Ejemplo
```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
NEXT_PUBLIC_APP_NAME=Punto de Venta
NEXT_PUBLIC_CLIENT_NAME=Ideas Graficas
NEXT_PUBLIC_SUPPORT_EMAIL=soporte@ideasgraficas.com
NEXT_PUBLIC_CURRENCY=MXN
NEXT_PUBLIC_TIMEZONE=America/Mexico_City
NEXT_PUBLIC_ENABLE_KITCHEN=true
NEXT_PUBLIC_ENABLE_INVENTORY=true
```
